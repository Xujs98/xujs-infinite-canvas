# 矩龙聚合平台服务器迁移维护公告

## 公告标题

矩龙聚合平台服务器迁移维护公告（7月21日 17:00-20:00）

## 公告摘要

矩龙聚合平台（julong-api）将于 2026年7月21日 17:00-20:00 进行服务器迁移。维护期间，聚合平台网站、API 服务及依赖该平台的矩龙画布相关功能将暂停服务。

## 公告正文

尊敬的用户：

为提升服务稳定性与后续承载能力，矩龙聚合平台（julong-api）计划于 **2026年7月21日 17:00-20:00（北京时间）** 进行服务器迁移和数据完整性校验。

平台地址：[https://api.julongkj.top/](https://api.julongkj.top/)

维护期间，以下服务将暂时不可用：

- 矩龙聚合平台网站登录及后台管理
- 图片、视频、音频及文本模型 API 调用
- 异步生成任务提交、状态轮询和结果获取
- 依赖聚合平台的矩龙画布 Web 端及 App 端功能
- 画布保存与同步、素材库和历史任务访问
- 订阅、签到及算力点相关操作

请在维护开始前保存当前画布，并避免在临近维护时提交耗时较长的生成任务。维护期间请勿重复提交请求，以免产生无效任务。

本次迁移预计于 **20:00** 完成。若迁移或数据校验提前结束，服务将提前恢复；如遇特殊情况，恢复时间可能顺延，我们会及时更新公告。

感谢您的理解与配合。

矩龙聚合平台（julong-api）
2026年7月21日
